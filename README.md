# frontend-boilerplate

Copy this repository, and you can start web frontend development instantly.

## How to use

To develop, run

    yarn start

To design components, run

    yarn storybook

## Libraries

### UI Components

* react
* styled-components
* radium

### State management

* redux
* redux-saga
* redux-persist
* redux-logger

### Routing

* react-router

### Linter

* eslint

### Build

* webpack

### Design

* storybook
